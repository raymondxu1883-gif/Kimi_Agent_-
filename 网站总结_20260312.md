# 欧美婚礼标识网站 - 项目总结

## 项目概述

这是一个专门为婚礼标识和装饰品设计的精美网站，旨在为即将结婚的情侣提供高质量的婚礼标识解决方案。网站采用现代、优雅的设计风格，突出展示各种婚礼标识产品，并提供便捷的咨询和定制服务。

## 技术栈

- **前端框架**：React 19 + TypeScript
- **构建工具**：Vite
- **样式方案**：Tailwind CSS + shadcn/ui 组件库
- **路由管理**：React Router（HashRouter）
- **图标库**：Lucide React
- **部署平台**：GitHub Pages
- **CI/CD**：GitHub Actions

## 页面结构

### 1. 首页 (`/`)
- 英雄区域（Hero Section）：展示品牌定位和核心价值
- 产品预览：精选产品展示
- 服务流程：展示服务步骤
- 作品展示：婚礼现场案例
- 客户评价：真实客户反馈
- 常见问题：FAQ 部分
- 联系表单：免费咨询

### 2. 产品页面 (`/products`)
- 英雄区域：产品分类介绍
- 搜索功能：快速查找产品
- 分类筛选：按产品类别过滤
- 排序功能：按价格、名称等排序
- 产品网格：卡片式展示所有产品
- 空状态处理：无搜索结果时的提示
- 行动召唤：引导用户咨询

### 3. 产品详情页 (`/products/:id`)
- 产品图片展示：主图和多角度预览
- 详细描述：产品特点和优势
- 特性列表：核心功能和参数
- 价格信息：清晰的价格展示
- 交付信息：生产时间和 shipping 详情
- 常见问题：产品相关问答
- 行动召唤：定制报价和咨询

### 4. 关于我们 (`/about`)
- 品牌故事：公司发展历程
- 团队介绍：核心成员展示
- 核心价值观：品牌理念和承诺
- 统计数据：业务成就和客户满意度

### 5. 联系我们 (`/contact`)
- 联系信息：电话、邮箱、地址
- 联系表单：详细的咨询表单
- 社交媒体：社交平台链接
- 服务时间：工作时间和响应时间
